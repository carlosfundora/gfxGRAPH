# gfxgraph-native

Native companion package for `gfxgraph`.

## Install

```bash
pip install /path/to/gfxGRAPH
pip install /path/to/gfxGRAPH/native
```

The first command installs the pure-Python package. The second builds and installs
`libhipgraph_bridge.so` into the companion package used by the runtime loader.
